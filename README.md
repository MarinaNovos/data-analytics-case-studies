# Data Analytics Case Studies

A collection of data analytics projects demonstrating exploratory data analysis, statistical reasoning, and data-driven insights.
The repository contains practical case studies using Python for data analysis and visualization.

## Tools

Python • Pandas • NumPy • Matplotlib • Seaborn • Jupyter

## Projects

### Customer Behaviour Analysis
Exploratory analysis of customer activity and engagement patterns.

### Product Usage Analysis
Investigation of user interaction data to identify trends and insights.

### Business Metrics Analysis
Analysis of key metrics to support data-driven decision making.

### Data Cleaning and Exploration
Techniques for preprocessing and exploring real-world datasets.

## Skills Demonstrated

- Exploratory Data Analysis (EDA)
- Data preprocessing
- Statistical analysis
- Data visualization
- Analytical problem solving
